##### （6）MVVM：是前端视图层的分层开发思想，主要把每个页面，分成M、V、VM，其中VM是MVVM思想的核心，因为VM是M和V之间的调度者。M：保存的是每个页面中单独的数据；V：是每个页面中的HTML结构；VM：是一个调度者，分割了M和V，每当V层想要获取后保存数据的时候，都要VM做中间的处理。前端页面中使用MVVM的思想，主要是为了让我们开发更加方便，因为MVVM提供了数据的双向绑定
