#### 1、定位

##### static：默认值，没有定位

##### relative：相对定位元素的定位是相对其正常位置

##### fixed：元素位置相对于浏览器窗口是固定位置

##### absolute：绝对定位的元素相对于最近的已定位父元素，如果元素没有父元素，则它相对于< html >

##### sticky：粘性定位，基于用户的滚动位置来定位
